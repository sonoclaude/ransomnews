Le API di ransomNews espongono in formato JSON i dati raccolti e verificati manualmente sulle rivendicazioni ransomware contro organizzazioni italiane, a partire dall'ottobre 2020. Accesso libero, nessuna autenticazione richiesta. The ransomNews API exposes in JSON format the manually collected and verified data on ransomware claims against Italian organizations, starting from October 2020. Open access, no authentication required.

GET rivendicazioni.json Tutti i recordAll records

Restituisce l'elenco completo di tutte le rivendicazioni ransomware verificate, dal 2020 ad oggi. Returns the complete list of all verified ransomware claims, from 2020 to date.

URL
https://ransomnews.online/api/v1/rivendicazioni.json

Note sull'utilizzoUsage notes
Aggiornamento automaticoAutomatic updates
I file JSON vengono rigenerati automaticamente ad ogni aggiornamento della dashboard tramite GitHub Actions. Il campo generated in ogni risposta indica la data e ora dell'ultima rigenerazione. JSON files are automatically regenerated on every dashboard update via GitHub Actions. The generated field in each response indicates the date and time of the last regeneration.
CORS
Le API sono servite tramite GitHub Pages e supportano richieste cross-origin (CORS). È possibile interrogarle direttamente da browser o da qualsiasi client HTTP. The APIs are served via GitHub Pages and support cross-origin requests (CORS). They can be queried directly from a browser or any HTTP client.
Dati verificati manualmenteManually verified data
Tutti i record sono verificati manualmente. I dati relativi agli anni 2020–2023 non includono i GB pubblicati (campo dati_gb = 0). Le rivendicazioni con vittima anonimizzata o rimosse dal data leak site sono incluse con le informazioni disponibili. All records are manually verified. Data for years 2020–2023 does not include published GB (dati_gb = 0). Claims with anonymized victims or removed from the data leak site are included with available information.
Utilizzo responsabileResponsible use
I dati sono forniti a scopo di ricerca, giornalismo e consapevolezza sulla minaccia ransomware. Si prega di citare ransomNews come fonte in caso di pubblicazione. Data is provided for research, journalism, and ransomware threat awareness purposes. Please cite ransomNews as the source when publishing.
